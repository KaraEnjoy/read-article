## 故事发展脉络

### 孙少安线
农村改革先行者 → 砖厂创业者 → 企业家 → 带领村民致富

### 孙少平线
知识青年 → 打工者 → 煤矿工人 → 精神成熟的劳动者

### 田润叶线
青梅竹马的爱情 → 现实的婚姻 → 婚姻危机 → 和解与成长

### 田福军线
基层干部 → 地区专员 → 省委常委 → 改革推动者

## 主题发展

1. **从困境到希望**：文革末期的困境到改革开放的希望
2. **从选择到成长**：人生选择到个人成长与成熟
3. **从个人到社会**：个人奋斗到社会发展的缩影
4. **从理想到现实**：理想主义到现实主义的平衡
5. **从平凡到伟大**：平凡生活中的伟大精神
