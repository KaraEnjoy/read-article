# 《红楼梦》故事梗概

## 一、作品基本信息

**作者**：曹雪芹（前八十回）、高鹗（后四十回，有争议）
**创作年代**：清代乾隆年间
**体裁**：长篇小说
**地位**：中国古典文学巅峰之作，四大名著之一

## 二、故事背景

《红楼梦》以清代康乾盛世为背景，以贾、史、王、薛四大家族的兴衰为线索，描绘了封建社会晚期的社会生活画卷。故事主要围绕贾宝玉、林黛玉、薛宝钗三人的爱情婚姻悲剧展开，同时展现了大观园中众多女性的命运遭遇。

## 三、核心人物关系

### 主要人物
- **贾宝玉**：贾府嫡孙，衔玉而生，性格叛逆，反对封建礼教
- **林黛玉**：贾母外孙女，才貌双全，多愁善感，与宝玉真心相爱
- **薛宝钗**：薛姨妈之女，端庄贤淑，深通世故，与宝玉有金玉良缘之说
- **贾母**：贾府最高权威，疼爱宝玉，支持宝黛爱情
- **王熙凤**：贾琏之妻，精明能干，心狠手辣，贾府实际管家
- **贾政**：宝玉之父，封建卫道士，严厉刻板
- **王夫人**：宝玉之母，表面慈善，实际冷酷
- **贾元春**：宝玉姐姐，皇妃，贾府靠山
- **贾探春**：贾政庶女，精明强干，有远见
- **史湘云**：贾母侄孙女，开朗豪爽
- **妙玉**：带发修行的尼姑，高洁孤僻
- **晴雯**：宝玉贴身丫鬟，聪明伶俐，反抗性强
- **袭人**：宝玉丫鬟，温柔贤惠，恪守封建礼教

## 四、故事主线

### 第一部分：荣国府的繁华
1. **贾宝玉出世**：神瑛侍者转世，衔通灵宝玉而生
2. **林黛玉进贾府**：母亲去世后投靠外祖母，与宝玉初次见面，似曾相识
3. **薛宝钗进京**：因选秀入住贾府，随身携带金锁，与宝玉的通灵玉形成"金玉良缘"之说
4. **大观园建成**：为元妃省亲而建，极尽奢华
5. **元妃省亲**：贾家达到鼎盛时期

### 第二部分：宝黛爱情的发展
1. **共读西厢**：宝黛在大观园共读《西厢记》，感情升温
2. **黛玉葬花**：黛玉感怀身世，葬花吟词，宝玉聆听后更知其心
3. **宝玉挨打**：因与戏子蒋玉菡交往及金钏儿之死，被贾政毒打
4. **宝钗探伤**：宝钗送药探伤，表现出对宝玉的关心
5. **黛玉题帕**：宝玉送旧帕，黛玉题诗，互诉衷肠

### 第三部分：贾府的衰落
1. **抄检大观园**：因绣春囊事件，王熙凤带人抄检大观园，许多丫鬟被逐
2. **晴雯之死**：因被王夫人认为狐媚惑主，被赶出贾府后病逝
3. **宝玉失玉**：通灵宝玉丢失，宝玉变得痴呆
4. **黛玉焚稿**：得知宝玉要娶宝钗，黛玉焚稿断情，含恨而死
5. **宝玉成亲**：王熙凤设计"掉包计"，宝玉误以为娶的是黛玉，实际娶了宝钗

### 第四部分：家族的彻底败落
1. **元妃薨逝**：贾府失去靠山
2. **贾府被抄**：因贾政外放期间的亏空及其他罪行，贾府被抄家
3. **贾母去世**：贾府最后一根精神支柱倒塌
4. **宝玉出家**：经历家族变故和黛玉之死，宝玉看破红尘，出家为僧
5. **宝钗守寡**：宝玉出家后，宝钗独自抚养儿子贾桂

## 五、主题思想

1. **爱情悲剧**：宝黛爱情因封建礼教的束缚而悲剧收场，反映了对自由爱情的追求
2. **家族兴衰**：四大家族的由盛转衰，揭示了封建社会必然走向灭亡的历史趋势
3. **女性命运**：大观园中众多女性的悲剧命运，批判了封建礼教对女性的压迫
4. **人性探索**：通过对人物性格的细腻刻画，展现了人性的复杂性
5. **生命意义**：通过宝玉的出家，探讨了生命的终极意义和超脱之路

## 六、艺术特色

1. **人物塑造**：塑造了数百个性格鲜明的人物形象，个个栩栩如生
2. **结构布局**：以宝黛爱情为主线，以四大家族兴衰为副线，结构严谨
3. **语言艺术**：语言优美典雅，诗词歌赋融入其中，具有高度的文学性
4. **象征手法**：大量运用象征手法，如通灵宝玉、绛珠仙草等，增强作品的深度
5. **社会批判**：对封建社会的种种弊端进行了深刻批判

## 七、重要情节

1. **木石前盟**：神瑛侍者与绛珠仙草的前世因缘
2. **刘姥姥进大观园**：通过刘姥姥的视角，展现了贾府的奢华生活
3. **海棠诗社**：大观园众姐妹结社赋诗，展现了女性的才华
4. **寿怡红群芳开夜宴**：宝玉生日夜宴，众人占花名，暗示各自命运
5. **查抄宁国府**：贾府彻底败落的标志

## 八、结局

宝玉在中举之后，毅然出家为僧。宝钗虽然怀有身孕，但只能独自抚养孩子。贾府在经历抄家之后，虽然有短暂的复兴迹象，但最终还是走向了衰落。"落了片白茫茫大地真干净"，象征着一切繁华终归于空。

《红楼梦》通过一个家族的兴衰，展现了封建社会的全貌，是一部百科全书式的文学巨著，具有永恒的艺术价值。
